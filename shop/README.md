# 购物车系统
## 使用教程
修改settings里面的数据库连接数据，在数据库中创建数据库  
管理员账号为admin,密码为admin  
用户账号可自行注册

### 用户操作
用户可以搜索商品添加到自己的数据库中，可以删除/清空购物车，可以修改购物车商品的数量  
### 管理员操作
管理员可以向系统中添加商品，删除商品。
